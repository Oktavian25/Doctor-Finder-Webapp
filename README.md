### Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

   ```bash
   npm install
   ```

   ```bash

   ```

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Restore .NET dependencies:
   ```bash
   dotnet restore
   ```

3. Run the application:
   ```bash
   dotnet run
   ```
